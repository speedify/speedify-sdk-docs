Introduction to Enhanced Statistics

Enhanced Statistics is a library that can be added to the Speedify SDK to provide additional statistics and information about the quality of the users internet connections.

To use Enhanced Statistics, you must first initialize the Speedify SDK and enable the Enhanced Statistics add-on library. See the Initializing the SDK page for more information.

Enhanced Statistics will work gathering extra stats in the background automatically. These stats will be periodically sent to the Speedify servers for analysis.

If you want location information, you must request fine location and background location permissions in your app. Enhanced Statistics will not request location permissions on its own.

if (!EnhancedStatistics.hasFineLocationPermission(context)) {
EnhancedStatistics.initializeFineLocationPermission(context);
}
if (!EnhancedStatistics.hasBackgroundLocationPermission(context)) {
EnhancedStatistics.initializeBackgroundLocationPermission(context);
}
