Speedify SDK for Android 16.7.0.1/ Guides/ Permissions

Required Permissions

  • INTERNET
    • Talk to speed servers on the internet
  • FOREGROUND_SERVICE
    • Ability to run VPN service in background
  • ACCESS_NETWORK_STATE, ACCESS_WIFI_STATE, CHANGE_NETWORK_STATE
    • Check and control network adapter state
  • FOREGROUND_SERVICE_SYSTEM_EXEMPTED
    • Ability to run VPN service in background as of Android U

Optional Permissions

  • RECEIVE_BOOT_COMPLETED
    • Automatic reconnect after boot
  • QUERY_ALL_PACKAGES
    • Better Streaming mode performance, Allow apps that use P2P for legal purposes to stay on normal servers
  • VIBRATE
    • Default foreground notification will vibrate when entering kill switch mode
  • CHANGE_WIFI_MULTICAST_STATE
    • Required for network sharing
  • POST_NOTIFICATIONS
    • Required to see the foreground notification on Android T+
