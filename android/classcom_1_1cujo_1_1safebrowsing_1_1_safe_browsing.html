◆ reportHomeNetwork() [1/2]

final void com.cujo.safebrowsing.SafeBrowsing.reportHomeNetwork ( )

Report the current connected network is a home network.

Uses default retry/delay/timeout values If current network is Safe Browsing protected, ON_PROTECTED_NETWORK will be returned from OnSafeBrowsingConnectError If current network is not protected, will return OK. May also return any other error type in case of error. A disconnect will not occur. In previous versions, this may cause a disconnect if on a protected network

Since
1.15

◆ start()

final void com.cujo.safebrowsing.SafeBrowsing.start ( @NonNull String initConfig)

Start Safe Browsing.

Wrapper for initWithConfig and setEnabled. After this call Safe Browsing is initialized and enabled. If initialized and enabled is available from OnSafeBrowsing. Errors initializing will be signalled in OnSafeBrowsingConnectError.
