Speedify SDK for Android  8.2.1
What's New

Speedify SDK for Android 8.2

  • Port forwarding has been added. Call setForwardPorts to forward ports on a dedicated server to the client device.

Speedify SDK for Android 8.1

  • Speedify SDK now depends on Android SDK 29 (Q) for compatibility with Android 10's new security changes.

Speedify SDK for Android 8.0.2

  • setExcludeDefaultPrivateIPRanges can be set to false to allow private IP ranges to be routed over the tunnel.

Speedify SDK for Android 7.6.1

  • Set other apps to not use the VPN with setNonVPNApplications

Speedify SDK for Android 7.5

  • The Speedify SDK will not automatically reconnect if the VPN was terminated abnormally while in a connected state. Override OnServiceRestart to stop this behavior.
  • OnNewAdapter now has a NetworkType argument.
  • refreshDirectoryDomain can be used to get the current directory domain via OnDirectoryDomain .
  • OnConnectedServer now sends a ServerInformationDetailed object with more information on the connect server..

Speedify SDK for Android 7.4

  • You can now set DNS servers used by the VPN via setDNSServers

Speedify SDK for Android 7.0

  • ConnectionStats now shows download and upload speeds, as well as download and upload maximum speed estimates for connections.
