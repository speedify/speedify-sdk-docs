Speedify SDK for Android 0.0.0.1/ Guides/ Residential Proxy Configuration

The Residential Proxy API instructs the currently connected Speed Server to put specific traffic from the configured ports through a proxy running on a residential IP address. This is useful for accessing content that may be blocked or restricted when accessed via data center IP addresses. By default, TCP and UDP ports 53, 80, and 443 are proxied when the residential proxy is enabled.

As on 16.3, only TCP traffic is supported for residential proxying. As of 16.5, UDP traffic is also supported.

Residential Proxy API

  • setResidentialProxyEnabled - Enables or disables the residential proxy.
  • setResidentialProxyPorts - Set ports to proxy.
